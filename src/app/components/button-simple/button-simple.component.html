title

Type Detail
string text value for button

buttonType

Button type is defined to background color of the button, default value primary

Value Detail
primary Background color and text color
secondary Background color and text color
tertiary Background color and text color
quaternary Background color and text color
error Background color and text color
success Background color and text color

disabled

Type Detail
'true' | 'false' | 'disabled' | boolean | "" default value false

height and width

Height and width is used to set static values, by default set width 100% and height 40px

value Type Detail Example
height string set style height, can be set px, %, em, etc height="40px"
width string set style width, can be set px, %, em, etc width="200px"
