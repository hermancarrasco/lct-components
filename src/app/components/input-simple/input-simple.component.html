title

title is up to input

Type Detail Example Example without title
string text value for title title="this is the title" title=""

placeholder

Type Detail Example
string text value for placeholder placeholder="this is the placeholder"

type

type is defined to input type, default value text

Value Detail
text set type to text, keyboard in mobile device is defined for each type configuration
email set type to email, keyboard in mobile device is defined for each type configuration
number set type to number, keyboard in mobile device is defined for each type configuration

disabled

Type Detail
'true' | 'false' | 'disabled' | boolean | "" default value false

pdaAutoEnter

If the value is true, when detect a key press, it emit enter after 200ms

Type Detail
boolean default value false

Events

Name Detail
(enterEmitted) Emitted when press enter or tab key, return input value
(iconClick) Emitted when press the icon if exists
(inputClick) Emitted when press the input
